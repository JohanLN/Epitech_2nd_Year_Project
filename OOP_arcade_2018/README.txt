L'arcade est un projet graphique de deuxième année qui a pour but d'apprendre à utiliser les librairies dynamiques.

Au lancement du jeu quelques minijeux seront jouable, mais la partie forte du projet est de le fait de pouvoir changer dynamiquement de jeu et/ou de librairie graphique.

Pour compiler le projet utilisez "make".

Pour lancer le jeu utilisez "./arcade <chemin_vers_une_librairie_graphique>". A noter que les librairies graphiques sont répertoriées dans le dossier libs/ et sont sous la forme d'un .so.

Une fois en jeu vous allez pouvoir changer de librairie graphique avec les touches "F1" pour la précédente et "F2" pour la suivante.

De même pour les jeux avec les touches "F3" et "F4".
